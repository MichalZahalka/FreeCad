# 2020 Aluminum Profile Projects

Grouped FreeCAD projects for parts that mount to a 2020 aluminum extrusion.

## Included Projects

- `vesa_2020_profile_mount` is a lightweight VESA mount for devices such as a mini PC.
- `ssd_holder` is a tray for the AXAGON EEM2-UG2 external SSD enclosure.

## Notes

- Each project keeps editable `FCStd` sources separate from printable exports stored in an `exports/` directory.
