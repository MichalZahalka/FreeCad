# SSD Holder For 2020 Aluminum Profile

FreeCAD source and print export for a tray that mounts to a 2020 aluminum extrusion and holds an AXAGON EEM2-UG2 SSD enclosure.

## Files

- `ssd_holder.FCStd` is the editable FreeCAD source model.
- `exports/ssd_holder.stl` is the current mesh export for printing.

## Notes

- Stored under `profile_2020/` together with other projects for 2020 aluminum profile.
